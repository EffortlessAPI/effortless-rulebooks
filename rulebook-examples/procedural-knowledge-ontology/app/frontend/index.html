PKO 2.0.0 Procedure Register

Live. Every value on this page is read from a vw_<entity> view in erb_procedural_knowledge_ontology, generated from procedural-knowledge-ontology-rulebook.json by rulebook-to-postgres. Calculated columns (actual_duration_minutes, is_late, assigned_role_label) are computed by Postgres from the rulebook formulas — this UI never re-derives them.
