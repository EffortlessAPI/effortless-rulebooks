# Narrative — How the Three-Valued Set Theory Works

A **set** is a **collection** of distinct objects, and like all naïve **sets** it is governed by **extensionality**: it is fully determined by its **members**, so **order** and **repetition** never matter. A **member** is any object that a set may **contain**, and because every member is itself a set, a set may even exhibit **self-membership**. **Comprehension** lets you form a set from any **condition** you can state; the **singleton** and the **empty-set** are the smallest examples this **definition** produces.

What makes this platform different is its treatment of **membership**. Classical naïve theory is **two-valued** and assumes **bivalence**: every membership question is a **predicate** that must come back **true** or **false**. This platform is instead **three-valued**. The third **truth-value** is **null** — meaning "does not resolve" — and it sits beside true and false as a first-class **semantics**, making membership **partial** rather than total. Wherever a **definition** does **converge**, the answer is **grounded** and the system behaves exactly like the **classical**, **bivalent** case; **null** appears only where a fact is **ungrounded** and **undefined**.

To evaluate compound conditions, the platform uses **Strong-Kleene** logic, also called **k3**. Each **connective** — **negation**, **conjunction**, **disjunction** — carries a **truth-table** describing how the third value propagates. The load-bearing fact is that **negation** maps null to null, which is exactly what lets a **self-referential** **loop** rest at null instead of exploding.

**Evaluation** itself is a grounding procedure borrowed from **Kripke**'s **fixed-point** construction. Every membership fact starts as null; the procedure promotes a fact to true or false only once its **condition** has already settled. A fact that never settles is **ungrounded** and stays null by the platform's missing **rule**, **rule-12**: when a definition's **loop** prevents a stable true-or-false value, the result is null. This is the **declarative** spirit of **cmcc** (the Conceptual Model Completeness Conjecture) — every **declarable** property has an outcome, and "does not converge" is a legitimate one.

The famous test case is the **russell-set**: the set of all sets that do not **contain** themselves. Asking whether it exhibits self-membership is the **russell-paradox**. In a **two-valued** world this is a flat **contradiction**: try true, contradiction; try false, contradiction. Under rule-12 the same trial lands on the only stable **fixed-point** — null. So the **russell-paradox** is not a **contradiction** at all but a single **ungrounded** membership fact. The set is a perfectly fine **definition** that produces one non-bivalent answer; nothing collapses because nothing forces **bivalence**.

This is not a new logic. It bolts standard moves from **partial** set theory, **paraconsistent** treatments of self-reference, and **k3** **semantics** directly into the informal **rule** list — choosing the "gap" (null) route where **paraconsistent** systems choose the "glut" route. The eleven classical rules survive untouched; **rule-12** simply adds the value every other self-referential system already uses.
